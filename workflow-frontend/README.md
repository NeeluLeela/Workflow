# Multi-Tenant Workflow System

## Tech Stack
Backend:
- Node.js
- Express
- PostgreSQL
- Zod
- JWT

Frontend:
- React
- Vite
- Axios
- React Router

## Setup

Backend
npm install
npm run dev

Frontend
npm install
npm run dev

## Environment
DATABASE_URL=postgresql://postgres:software@localhost:5432/workflow_db
JWT_SECRET=supersecret
